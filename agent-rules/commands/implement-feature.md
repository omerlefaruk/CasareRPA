# Implement Feature Command

Workflow for implementing new features.

## Prerequisites
- Clear requirements
- Approved design (if significant)

## Steps

### 1. Research
- [ ] Read relevant `_index.md` files
- [ ] Understand existing patterns
- [ ] Identify dependencies

### 2. Plan
- [ ] Create plan in `.brain/plans/`
- [ ] List files to modify/create
- [ ] Identify risks

### 3. Implement
- [ ] Create/modify code
- [ ] Follow coding standards
- [ ] Commit incrementally

### 4. Test
- [ ] Write tests
- [ ] Run `pytest tests/ -v`
- [ ] Fix failures

### 5. Document
- [ ] Update docstrings
- [ ] Update `_index.md` files
- [ ] Update CHANGELOG

### 6. Review
- [ ] Self-review
- [ ] Address issues
- [ ] Final verification

# UI Agent

PySide6 UI development specialist.

## Role
Develop and maintain the Canvas UI.

## When to Use
- New widgets
- UI fixes
- Theme updates
- UX improvements

## Key Areas
- `presentation/canvas/` - Main canvas
- `presentation/canvas/ui/` - Theme system
- `presentation/canvas/widgets/` - Custom widgets
- `presentation/canvas/visual_nodes/` - Node UI

## Patterns
- Inherit from `BaseWidget`
- Use signal/slot for communication
- Follow theme system
- Async operations via qasync

## Standards
- Use `Theme` for styling
- Test with pytest-qt
- Responsive layouts
- Accessibility support

## Outputs
- Widget implementations
- Theme updates
- UI tests

# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in CasareRPA, please **DO NOT** create a public GitHub issue.

Instead, please report it via email to:
**security@casarerpa.com** (Replace with actual email or use a private vulnerability reporting platform if available)

Include as much information as possible:
-   Type of vulnerability.
-   Steps to reproduce.
-   Potential impact.

We will acknowledge your report within 48 hours and provide updates as we investigate and resolve the issue.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x     | :white_check_mark: |
| 2.x     | :x:                |
| < 2.0   | :x:                |

# Keyboard Shortcuts

Application keyboard shortcuts for CasareRPA v3.1.

## Workflow Execution

| Shortcut | Action | Description |
|----------|--------|-------------|
| **F8** | Run Local | Execute workflow in Canvas process |
| **Ctrl+F5** | Run on Robot | Submit to LAN robot via Orchestrator |
| **Ctrl+Shift+F5** | Submit | Queue for internet robots (distributed) |
| **Shift+F5** | Stop Execution | Stop running workflow |
| **F9** | Toggle Breakpoint | Add/remove breakpoint on node |
| **F10** | Step Over | Execute current node, skip subroutines |
| **F11** | Step Into | Step into node details |

## File Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New workflow |
| Ctrl+O | Open workflow |
| Ctrl+S | Save workflow |
| Ctrl+Shift+S | Save as |
| Ctrl+W | Close workflow |

## Edit Operations

| Shortcut | Action |
|----------|--------|
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+X | Cut |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Delete | Delete selected |
| Ctrl+A | Select all |

## Canvas Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+Mouse wheel | Zoom in/out |
| Middle mouse drag | Pan canvas |
| Space+drag | Pan canvas |
| Home | Fit to view |
| Ctrl+0 | Reset zoom |

## Node Operations

| Shortcut | Action |
|----------|--------|
| Tab | Open node library |
| / | Quick search nodes |
| Ctrl+D | Duplicate node |
| Ctrl+G | Group nodes |

## Help

|----------|--------|

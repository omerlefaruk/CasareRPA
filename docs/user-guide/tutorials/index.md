## Prerequisites

Before starting these tutorials, ensure you have:

- CasareRPA installed and running
- Basic understanding of the Canvas interface (see [First Workflow](../getting-started/first-workflow.md))
- Internet connection (for web and API tutorials)
- Windows OS (for desktop automation)

## Learning Path

### Beginner (New to RPA)

```
First Workflow -> Browser Automation -> Data Processing
```

### Intermediate (Building Production Workflows)

```
Email Processing -> API Integration -> Scheduled Workflows
```

### Advanced (Enterprise Automation)

```

```
